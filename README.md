# SmartRoadGuardian

AI road safety monitoring system that detects potholes, road hazards, and unsafe conditions using mobile device cameras and sensors.

## Features

- Pothole detection from camera feed
- Accelerometer-based road roughness measurement
- Hazard geolocation and mapping
- Real-time alerts for detected hazards
- Community reporting and data aggregation

## Tech Stack

Kotlin, TensorFlow Lite, Android, Google Maps API, Firebase

## Getting Started

### Prerequisites

- Android Studio
- Gradle

### Installation

```bash
git clone https://github.com/karthik-idikuda/SmartRoadGuardian.git
cd SmartRoadGuardian
Open in Android Studio and sync Gradle.
```

### Usage

```bash
Build and run on an Android device.
```
